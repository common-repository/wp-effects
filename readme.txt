== Description ==

This effects plugin allows you to display cool snowflake effects on your wordpress blog, as well as leaves and your own custom images!

All you need to do is change the type of effect you wish to display through the WP Effects settings and then everything should be working fine!


== Installation ==

1. Upload `/wp-effects/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the settings in your Administration Panel
4. Everything should work.
